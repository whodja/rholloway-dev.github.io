Flappy Bird Clone
This project involved working in app deveolopment to a create a clone of the popular moblie game, Flappy Bird. Through several months dedicated towards in-game physics and user interface, this clone was made to mimic the playability of the beloved game that no longer exists for purchase. It was created through AndroidStudio using JavaScript, C++, and Kotlin.

Dungeon Game
This project involved making an indepth dungeon RPG Game. In it, the user takes the role of a hero seeking to fight their way through the dungeon to find the treasure. As they progress, they will come face to face with evil zombies, skeletons, and other creatures who will try to defeat them. At any time, they may access the shop to purchase better weapons, potions, and tools to fight. This was entirely coded using Python.

Cards Game
This porject involved creating a card game for the user to play. In it, the user must pick any card within a 52 card deck. They will then go against the CPU to see if they had the same as a random card that was already pulled. This program was made entirely using C++.
